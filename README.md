# 🚣 Rameur Trainer

Application web simple pour s'entraîner au **rameur indoor** lorsque l'écran du rameur est cassé ou absent.

👉 Utilisable directement depuis un navigateur sur **smartphone, tablette ou ordinateur**.

🔗 **Application en ligne :**
https://superoroborus.github.io/rameur-trainer/

---

# 🎯 Objectif du projet

Cette application a été créée pour permettre de continuer à utiliser un rameur dont le **moniteur de contrôle ne fonctionne plus**, tout en gardant :

* une cadence cible (SPM)
* un métronome sonore
* un programme d'entraînement structuré
* un chronomètre
* une visualisation du cycle de rame (drive / catch)

Elle peut également servir de **coach simple pour l'entraînement cardio au rameur**.

---

# ⚙️ Fonctionnalités

## 🗓 Programme d'entraînement sur 7 jours

Chaque jour propose une séance différente :

* endurance
* intervalles
* tempo
* sprint
* récupération active

Cela permet de varier les intensités au cours de la semaine.

---

## ⏱ Cadence cible (SPM)

L'application affiche une **cadence de rame recommandée** :

* 20 SPM
* 22 SPM
* 24 SPM
* 26 SPM
* 28 SPM

L'utilisateur doit simplement adapter sa vitesse de rame pour suivre le rythme.

---

## 🔊 Métronome sonore

Un **bip sonore synchronisé avec la cadence** aide à maintenir un rythme régulier.

Chaque bip correspond au **début du "drive"** (phase de poussée).

---

## 📊 Visualisation du coup de rame

Une barre animée indique les phases :

* **Drive** (poussée)
* **Catch / Recovery** (retour)

Cela permet de garder une cadence visuelle pendant l'entraînement.

---

## 📱 Compatible mobile et tablette

L'application fonctionne sur :

* smartphone
* tablette
* ordinateur

Elle peut être ouverte directement dans un navigateur :

* Safari
* Chrome
* Firefox
* Edge

---

# 🚀 Utilisation

1. Ouvrir l'application dans le navigateur
2. Sélectionner la séance du jour
3. Appuyer sur **Start**
4. Suivre la cadence indiquée
5. Utiliser **Pause** ou **Reset** si nécessaire

Parce que jeter un rameur fonctionnel juste à cause d'un écran cassé serait dommage.
